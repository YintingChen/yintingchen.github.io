1 - 定义 function #

A function \(f\) is a rule that
      assigns exactly one element \(f(x) \in E\)
      to each element \(x \in D\).

换句话说，如果同一个 \(x\) 对应了两个 \(y\) 值，那就不是 function。

2 - 三种特殊 functions #

1 - Piecewise defined functions
\(f(x) = \begin{cases} 1-x &\text{if } x\le-1 \newline x^2 &\text{if } x>-1 \end{cases} \)

2 - Even and odd functions
For every x in the domain,
      even function: \(f(-x) = f(x)\)
      odd function: \(f(-x) = -f(x)\)

3 - Increasing and decreasing functions
A function \(f\)
is increasing on an interval \(I\)
      if \(f(x_{1})<f(x_{2})\), whenever \(x_{1} < x_{2}\) in \(I\)
is decreasing on \(I\)
      if ……

3 - Mathematical models #

所谓的 Mathematical models 实际上就是一些常用的 functions。
这些常用的 functions 分为两种，一种叫做 Algebraic functions，一种叫做 Transcendental functions。

3.1 Algebraic functions #

本质上就是对 Polynomials 的加、减、乘、除、开根。比如：
\(g(x)=\frac{x^4-16x^2}{x+\sqrt{x}}+(x-2)\sqrt[3]{x^2+1}\)

有4种非常常用的 Algebraic functions，我们特别给它们取了名字。

（1）Linear functions
\(y = f(x) = mx + b\)
The point-slope form: \(y - y_1 = m(x - x_1)\)

（2）Polynomials
\(P(x) = a_{n}x^n+a_{n-1}x^{n-1}+…+a_{2}x^2+a_{1}x^1+a_{0}x^0\)
n must be a nonnegative integer，也就是说 n 不能是分数，也就是说不能出现\(\sqrt{x}\)的情况。

（3）Power functions
\(f(x)=x^a\)

熟悉这个结构的不同书写方式：
5叫做 base, 右上角的小数字叫做 exponent
\(5^{1/2}=\sqrt{5}\)
\(5^{3/8}=\sqrt[8]{5^3}\)
\(5^{-1}=\frac{1}{5}\)
\(5^{-2}=\frac{1}{5^2}\)

（4）Rational functions
\(f(x)=\frac{P(x)}{Q(x)}\)
P and Q must be polynomials and
the domain of f consists of all x such that \(Q(x)\not= 0 \)
\(h(x) = \frac{1+x}{1-\sqrt{x}}\) is not a rational function because the denominator is not a polynomial。

3.2 Transcendental functions #

总共分为3种。

（1）Trigonometric functions 三角函数
\(\sin (x+2\pi) = \sin x\)，\(-1\le\sin x\le1\)
\(\cos (x+2\pi) = \sin x\)，\(-1\le\cos x\le1\)
\(\tan x = \frac{\sin x}{\cos x}\)

\(\csc x = \frac{1}{\sin x}\) \(\sec x = \frac{1}{\cos x}\) \(\cot x = \frac{1}{\tan x}\)

\(\sin^{2} x + \cos^{2} x = 1 = (\sin x)^2 + (\cos x)^2\)

\(\sin (x+y) = \sin x\cos y + \cos x\sin y\)
\(\cos (x+y) = \cos x\cos y - \sin x\sin y\)

（2）Exponential functions
\(f(x) = b^x\)
b is called the base of \(f\) and muse be a positive constant.
x is called the exponent.

熟悉这个结构的不同书写方式：
5叫做 base, 右上角的小数字叫做 exponent
\(5^{1/2}=\sqrt{5}\)
\(5^{3/8}=\sqrt[8]{5^3}\)
\(5^{-1}=\frac{1}{5}\)
\(5^{-2}=\frac{1}{5^2}\)

Laws of exponents:
if a and b are positive numbers and
x and y are any real numbers, then
\(b^{x+y} = b^xb^y\)
\(b^{x-y} = \frac{b^x}{b^y}\)
\((b^x)^y = b^{xy}\)
\((ab)^x = a^xb^x\)

\(e \approx 2.71828\)
Natural exponential function: \(f(x) = e^x\)
The slope of the tangent line to \(y = e^x\) at (0,1) is exactly 1.

（3）Logarithmic functions
\(f(x) = \log_b x\)
x must be \(> 0\)
b is called the base of \(f\) and muse be a positive constant.\

We call the logarithmic functions the inverse functions of the exponential functions.
\(\log_b x = y \iff b^y = x\)

Laws of logarithms:
if x and y are positive numbers, then
\(\log_b{xy} = \log_b{x} + log_b{y}\)
\(log_b{(\frac{x}{y})} = \log_b{x} - log_b{y}\)
\(log_b{(x^r)} = r\log_b{x}\)

Natural logarithms: \(log_e{x}\) 常常写为 \(\ln{x} \)
\(\ln{x} = y \iff e^y = x\)
\(\ln{e} = \log_e{e} = 1\)
\(x^r = x^r\log_e{e} = e^{log_e{x^r}}\)

4 Build new functions from old functions #

4.1 Transformations of functions #

（1）Vertical and horizontal shifts
Suppose \(c > 0\)，
\(y = f(x) + c\)    [upward]
\(y = f(x) - c\)    [downward]
\(y = f(x - c)\)    [to the right]
\(y = f(x + c)\)    [to the left]

（2）Vertical and horizontal stretching
Suppose \(c > 1\)，
\(y = cf(x)\)    [stretch vertically]
\(y = (1/c)f(x)\)    [shrink vertically]
\(y = f(cx)\)    [shrink horizontally]
\(y = f(x/c)\)    [stretch horizontally]

（3）Vertical and horizontal reflecting
\(y = -f(x)\)    [reflect about x-axis]
\(y = f(-x)\)    [reflect about y-axis]

4.2 Combination of functions #

（1）加、减、乘、除
\((f+g)(x)=f(x)+g(x)\)
\((f-g)(x)=f(x)-g(x)\)
\((fg)(x)=f(x)g(x)\)
\((\frac{f}{g})(x)=\frac{f(x)}{g(x)}\)

（2）Composition of \(f\) and \(g\)
\((f\circ g)(x) = f(g(x))\)
The domain of \(f\circ g\) is the set of all x in the domain of g such that \(g(x)\) is in the domain of \(f\).
\((f\circ g\circ h)(x) = f(g(h(x)))\)

5 - Inverse functions #

A one-to-one function:
\(f(x_1) \not = f(x_2)\)    when \(x_1 \not = x_2\)

Let \(f\) be a one-to-one function wiht domain A and range B.
Then its inverse function \(f^{-1}\) has domain B and range A. \(f^{-1}(y)=x \iff f(x) = y\),   for any y in B

区分:
\(f^{-1}(x)\) 是 Inverse function of \(f(x)\)
\([f(x)]^{-1} = \frac{1}{f(x)}\)

Example. Find the inverse function of \(f(x)=x^3+2\)
step-1： \(y = x^3 + 2\)
step-2： \(x^3=y-2,    x=\sqrt[3]{y-2}\)
step-3： \(y = \sqrt[3]{x-2}\)
==> \(f^{-1}=\sqrt[3]{x-2}\)

The graph of \(f^{-1}\) is obtained by reflecting the graph of \(f\) about the line \(y = x\).
